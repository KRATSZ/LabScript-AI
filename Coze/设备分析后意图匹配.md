# 设备分析后意图匹配

## 🎯 意图分类体系

### A. OT-2设备控制意图 (OT-2 Device Control)
- OT-2设备控制。
- 用户需要直接操作OT-2机器人执行实验。
- 涉及设备启动、停止、监控、调试等OT-2专用操作。

### B. Flex设备控制意图 (Flex Device Control)
- Flex设备控制。
- 用户需要直接操作Flex机器人执行实验。
- 涉及设备启动、停止、监控、调试等Flex专用操作。

## 📋 意图匹配算法

```python
# 设备分析后意图匹配逻辑
def match_device_control_intent(device_type: str, user_input: str) -> str:
    """
    根据设备类型识别结果进行意图匹配
    """

    # 设备类型特征匹配
    ot2_features = [
        "p300_single_gen2", "p300_multi_gen2", "p1000_single_gen2", "p20_single_gen2",
        "opentrons_96_tiprack_300ul", "opentrons_96_tiprack_10ul",
        "temperaturemodule", "magneticmodule", "thermocycler"
    ]

    flex_features = [
        "flex_1channel_1000", "flex_8channel_1000", "flex_1channel_50", "flex_8channel_50",
        "opentrons_flex_96_tiprack_1000ul_hrf", "opentrons_flex_96_tiprack_200ul_hrf",
        "flex_heating_shaker", "flex_thermocycler", "flex_magnetic_block"
    ]

    # 控制操作关键词
    control_keywords = ["启动", "停止", "暂停", "继续", "重启", "复位", "校准", "查看状态", "检查", "调试"]

    # 意图匹配决策
    if device_type == "ot-2" and any(keyword in user_input for keyword in control_keywords):
        return "ot2_device_control"
    elif device_type == "flex" and any(keyword in user_input for keyword in control_keywords):
        return "flex_device_control"
    else:
        return "unclear"
```
