## 🧠 意图识别逻辑

### 意图分类体系

#### A. 日常闲聊 (Normal Chat)
- 日常闲聊。
- 问日期、时间（如"今天几号""现在几点"）一律归日常闲聊，不管上下文是否提到实验。
- 若不清楚，优先归日常闲聊。

#### B. 希望结束聊天 (End Chat)
- 用户希望结束对话。
- 例如"闭嘴""退下""别说了"。

#### C. Opentrons设备控制意图 (Device Control)
- Opentrons设备控制。
- 用户需要直接操作机器人执行实验。
- 涉及设备启动、停止、监控、调试等操作。

#### D. Opentrons代码生成意图 (Code Generation)
- Opentrons代码生成。
- 用户需要实验协议和代码。
- 涉及PCR、DNA、细胞培养等实验内容。

### 智能识别算法

```python
# 意图识别决策逻辑
def detect_user_intent(user_input: str) -> str:
    """
    多层次意图识别算法
    """

    # 关键词权重计算
    normal_keywords = ["你好", "帮助", "介绍", "说明", "解释", "咨询", "功能", "特性", "日期", "时间", "几号", "几点"]
    end_chat_keywords = ["闭嘴", "退下", "别说了", "停止", "结束", "再见", "拜拜"]
    device_control_keywords = [
        "启动", "开始", "运行", "执行", "停止", "暂停", "继续", "重启", "复位", "校准",
        "监控", "查看状态", "检查", "调试", "连接", "断开", "初始化", "清空", "清洗",
        "机器人状态", "设备状态", "运行状态", "停止运行", "开始运行"
    ]
    code_keywords = [
        "PCR", "DNA", "RNA", "细胞", "ELISA", "移液", "稀释", "混合", "培养", "提取",
        "Opentrons", "机器人", "移液器", "96孔板", "Tip头", "Flex", "OT-2",
        "协议", "自动化", "代码生成", "SOP", "实验步骤", "编程",
        "Python", "API", "模拟", "验证", "调试", "配置"
    ]

    normal_score = sum(1 for keyword in normal_keywords if keyword in user_input)
    end_chat_score = sum(1 for keyword in end_chat_keywords if keyword in user_input)
    device_control_score = sum(1 for keyword in device_control_keywords if keyword in user_input)
    code_score = sum(1 for keyword in code_keywords if keyword in user_input)

    # 决策逻辑（优先级规则）
    # 1. 结束聊天关键词检测（最高优先级）
    if end_chat_score > 0:
        return "end_chat"
    # 2. 日期时间问题一律归为日常闲聊
    elif any(time_keyword in user_input for time_keyword in ["日期", "时间", "几号", "几点", "今天", "现在"]):
        return "normal_chat"
    # 3. 设备控制关键词判断
    elif device_control_score > 0:
        return "device_control"
    # 4. 实验相关关键词判断
    elif code_score > 0:
        return "code_generation"
    # 5. 日常闲聊判断
    elif normal_score > 0 or len(user_input) < 10:
        return "normal_chat"
    # 6. 不明确情况优先归日常闲聊
    else:
        return "normal_chat"  # 按规则要求，不清楚时优先归日常闲聊
```

### 意图识别规则

#### 正常对话意图特征
- 问候类：你好、hi、早上好等
- 咨询类：介绍、说明、解释、帮助等
- 功能类：功能、特性、支持、如何等
- 简短查询：通常少于10个字符的简短输入

#### 设备控制意图特征
- 操作命令：启动、停止、暂停、继续、重启、复位、校准等
- 状态查询：查看状态、检查连接、监控进度、调试等
- 实时控制：执行、运行、开始、中断等操作性词汇
- 设备交互：直接针对机器人硬件的操作指令

#### 代码生成意图特征
- 实验类型：PCR、DNA提取、细胞培养、ELISA等
- 设备相关：Opentrons、机器人、移液器、96孔板等
- 技术术语：协议、自动化、Python、API、模拟等
- 操作描述：生成、编程、配置、验证等
