# Opentrons脚本快速验证工具

## 安装说明
将 `opentrons_validator` 文件夹放在你的项目根目录下。

## 快速验证脚本
```bash
cd opentrons_validator
python simulate.py your_script.py
```

## 示例
```bash
# 从opentrons_validator文件夹内运行
cd opentrons_validator
python simulate.py "../论文中要用的实验脚本案例/Bradford_proteinassay.py"

# 或者验证当前目录的脚本
python simulate.py ../your_script.py
```

## 在Cursor中使用

### 方法1: 直接在终端运行
1. 打开Cursor终端
2. 运行: 
   ```bash
   cd opentrons_validator
   python simulate.py ../当前文件名.py
   ```

### 方法2: 设置快捷键 (推荐)
在Cursor中添加自定义任务：

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "Tasks: Configure Task"
3. 添加以下配置到 `.vscode/tasks.json`:

```json
{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "验证Opentrons脚本",
            "type": "shell",
            "command": "python",
            "args": ["opentrons_validator/simulate.py", "${file}"],
            "group": "build",
            "presentation": {
                "echo": true,
                "reveal": "always",
                "focus": false,
                "panel": "new"
            }
        }
    ]
}
```

4. 使用快捷键 `Ctrl+Shift+P` → "Tasks: Run Task" → "验证Opentrons脚本"

### 方法3: 创建便捷脚本 (Windows)
在项目根目录创建 `validate.bat`:
```batch
@echo off
cd opentrons_validator
python simulate.py %1
```

然后可以直接运行: `validate.bat your_script.py`

## 让AI自动修复脚本

复制 `自动修复脚本提示词.md` 的内容，然后：

1. 运行验证命令得到错误信息
2. 将错误信息和脚本内容一起发给AI
3. AI会自动修复脚本
4. 保存修复后的脚本并重新验证

## 工作流程
1. 编写/修改Opentrons脚本 
2. 进入 `opentrons_validator` 文件夹
3. 运行 `python simulate.py ../脚本名.py` 验证
4. 如有错误，复制错误信息给AI修复
5. 重复3-4直到验证通过
